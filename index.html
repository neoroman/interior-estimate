Meeting Prep Tool

견적봄

공정별 자재 수준을 고르면 예상 견적 범위와 미팅 질문 목록을 바로 확인할 수 있습니다.

이 견적은 참고용이며 실제 금액은 현장 조건에 따라 다릅니다. 정확한 금액은 전문가에게 확인하세요.

CALCULATOR

견적 범위 계산

15개 공정에서 실제 적용할 공정만 켜고, 자재 수준과 받은 단가를 같이 넣어보세요.

예상 총액 범위

지역·현장 조건에 따라 변동될 수 있습니다.

평

만원

비워두면 적정/비쌈/싸다 판단은 생략됩니다.

현재 선택한 자재

    공정별 단가 판정

    단가를 입력하면 자재 기준 범위와 비교한 판정이 표시됩니다.

      CHECK

      공정 누락 체크리스트

      선택한 공정과 붙여넣은 견적서 텍스트를 기준으로, 이 견적에 보통 들어가야 할 항목을 자동 점검합니다.

      공정을 선택하면 등급을 몰라도 먼저 확인해야 할 기본 항목을 자동으로 정리합니다.

      PREP

      상담 준비

      이번 미팅에서 먼저 물어볼 핵심 항목, 공통 체크리스트, 공정별 상세 질문을 한 곳에서 정리합니다.

      이번 미팅에서 꼭 확인할 7가지

      선택한 공정, 자재 기준, 입력 단가, 총액, 견적서 텍스트를 합쳐 우선순위가 높은 항목부터 보여줍니다.

      조건을 입력하면 이번 미팅에서 먼저 확인해야 할 포인트를 자동 요약합니다.
      추가 확인사항 더 보기
      핵심 항목 외에 더 확인해두면 좋은 내용이 여기에 정리됩니다.

      빠른 체크리스트

      업체와 상담할 때 거의 항상 확인해야 하는 공통 항목입니다.

      • 각 공정에서 어떤 등급과 브랜드의 자재를 쓰는지 물어보기
      • 시공비와 자재비를 분리한 항목별 견적서 요청하기
      • 자재값 인상 주장 시 어떤 자재가 언제 얼마나 올랐는지 확인하기
      • 등급을 한 단계 낮췄을 때 절감되는 금액 질문하기
      • 철거, 전기, 설비 금액을 각각 분리해서 요청하기
      • 보증 범위와 하자 처리 기간 확인하기
      • 견적서에 적힌 브랜드와 모델명이 실제 시공 제품과 동일한지 확인하기
      • 변동 가능 항목은 무엇이고, 변경 시 어떤 방식으로 추가 견적서를 주는지 확인하기

      디자인 미팅 때 참고할 생활 체크

      실측 현장에서 다 보기 어려운 내용은 디자인 미팅에서 생활 방식 기준으로 정리하는 편이 더 정확합니다.

      • 침대, 소파, 식탁, 냉장고, 세탁기 같은 큰 가구·가전이 들어왔을 때 답답하지 않은 배치인지 이야기하기
      • 현관문, 방문, 중문, 냉장고 문, 붙박이장 문, 욕실문이 서로 간섭 없게 열리도록 요청하기
      • 침대 옆, 소파 옆, 식탁 주변, 청소기 자리, 정수기 자리의 콘센트와 스위치 위치를 생활 기준으로 정하기
      • 신발장, 팬트리, 주방 서랍, 욕실장, 세탁실 선반이 내 물건 기준으로 충분한지 먼저 설명하기
      • 싱크 높이, 조리대 폭, 상부장 높이처럼 매일 쓰는 주방 기준을 내 키와 습관에 맞춰 요청하기
      • 세면대 높이, 샤워 공간, 수건걸이, 휴지걸이, 욕실장 위치를 실제 사용 동선 중심으로 맞춰달라고 요청하기
      • 거실, 침실, 주방 조명을 한 번에 켤지 나눠 켤지, 간접조명이 필요한지 생활 패턴 기준으로 정하기
      • 분전반, 인터넷 단자함, 보일러, 배관, 점검구 중 어디까지 가리고 어디는 열어둘지 미리 결정하기
      • 청소가 쉬운 마감인지, 먼지·물때·손자국이 잘 남지 않는지 유지관리 관점으로 자재를 고르기

      공정별 상세 질문

      선택한 공정과 등급 기준에 따라 바로 꺼내 쓸 수 있는 세부 질문입니다.

      SITE CHECK

      말 안하면 빠지는 시공 체크리스트

      현관, 거실, 주방, 욕실에서 시공 전에 먼저 말해둬야 하는 항목을 정리했습니다.

      공정 선택과 상관없이 자주 놓치는 항목을 공간별로 확인할 수 있습니다.

      현관

      처음 들어올 때 체감이 큰 조명·수납·마감 포인트입니다.

      공통 체크
      1. 조명 스위치 회로 분리. 어떤 조명을 묶을지 미리 정하기
      2. 현관 주변부 마감 방식 확인
      3. 일괄 소등 스위치 넣기
      4. 우유 투입구 막기 여부 확인
      5. 신발장 간접 조명 여부 확인
      6. 신발장 선반은 줄다보 찬넬처럼 이동 가능한 형식으로 가능한지 확인

      거실

      배선, 수평·수직, 바닥, 커튼박스처럼 시공 전 결정해야 하는 항목들입니다.

      공통 체크
      1. 콘센트, 인터넷 선 위치, 스위치, 기초 배선 위치 미리 확정
      2. 조명 회로 분리. 스위치를 적절히 나누기
      3. 벽면 기초 작업, 목공 기초 작업 수직·수평 맞추기
      4. 거실·주방·공용부 벽체와 붙박이 가구 들어가는 부분 수평 확인
      5. 바닥 셀프 레벨링 여부 확인
      6. 마루 시공 방향 미리 정하기
      7. 커튼 박스 크기 적절하게 확보하기. 20cm 기준 확인
      8. 우수관에 배관 단열재 감싸기

      주방

      치수와 빌트인 공간을 먼저 고정하지 않으면 나중에 수정이 어렵습니다.

      공통 체크
      1. 싱크대 높이 880~900mm 범위에서 사용자에게 맞는 높이 확인
      2. 빌트인 가전 공간 확보 여부 확인
      3. 음식물 처리기, 싱크볼, 하부장, 정수기 들어갈 자리까지 같이 체크
      4. 서랍 구성 포함 주방 수납 구성도 미리 소통

      욕실

      타일 시작점, 설비 위치, 수납과 조명 위치는 시공 전에 안 잡으면 되돌리기 어렵습니다.

      공통 체크
      1. 샤워 부스 파티션 여부 확인
      2. 세면대 높이 850~870mm 범위에서 편한 높이 직접 재보기
      3. 욕실 콘센트가 보이지 않게 설계 가능한지 확인. 욕실장 안쪽 배치 검토
      4. 센서등은 주 동선 위쪽으로 설치할지 설계 단계에서 소통
      5. 샤워볼 걸이 위치 확인
      6. 타일 시작점 체크
      7. 위시타워장 여부 확인
      8. 발코니 단열 확인

      내 체크리스트 추가

      사람마다 필요한 확인 항목을 공간이나 제품 기준으로 따로 저장해두고 계속 관리할 수 있습니다.

      아직 저장한 개인 체크리스트가 없습니다.

      MEASUREMENT

      실측 체크리스트

      실측하러 갈 때 현장에서 바로 확인해야 하는 치수, 간섭, 설비 포인트를 공간별로 정리했습니다.

      실측 전 준비물부터 공간별 핵심 치수까지, 현장에서 빠뜨리기 쉬운 항목을 먼저 확인하세요.

      SPEC

      스펙 · 용어 사전

      15개 공정을 기준으로 스펙과 용어를 더 읽기 쉽게 묶었습니다. 예를 들어 창호는 로이유리, 단열간봉, 시스템창호처럼 한 카테고리 안에서 같이 볼 수 있습니다.

      공정별 핵심 스펙

      현재 선택한 공정과 세부 기준을 바탕으로, 상담 전에 먼저 이해해두면 좋은 핵심 스펙을 공정별로 묶어서 보여줍니다.

      공정별 용어 사전

      견적서에 자주 나오는 표현을 공정별로 나눠서 보고, 바로 물어볼 질문까지 같이 준비하세요.

      선택한 공정 기준으로 관련 용어를 먼저 보여줍니다.

      NEGOTIATION

      협상 시뮬레이터

      공정별로 유지, 한 단계 낮추기, 제외 검토를 바꿔보며 예상 중간 총액 차이와 바로 쓸 협상 문장을 따로 확인할 수 있습니다.

      추천 협상안 3개

      현재 선택 기준에서 절감 효과가 큰 순서대로 바로 시도해볼 만한 협상안을 먼저 정리합니다.

      총액 변화

      등급 조정이나 제외 검토를 적용했을 때 현재 기준과 얼마나 차이 나는지 바로 보여줍니다.

      현재 기준 중간 총액

      -

      현재 선택한 세부 기준의 중간값 합산입니다.

      시뮬레이션 총액

      -

      아래 시나리오를 적용했을 때의 예상 중간 총액입니다.

      예상 차이

      -

      공정을 선택하면 절감 또는 증가 포인트를 자동으로 요약합니다.

      공정별 시나리오 조정

      절감 효과가 큰 공정부터 먼저 보이도록 정렬됩니다. 한 단계 낮추기 또는 제외 검토를 적용해보고, 빠지는 스펙을 바로 확인하세요.

      협상할 때 바로 쓸 문장

      시뮬레이션에서 바뀐 항목을 기준으로 바로 꺼내 쓸 수 있는 질문입니다.
