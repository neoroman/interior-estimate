# 견적봄 해커톤 발표 구성

1. Cover
- 문제 한 줄: 인테리어 견적서는 복잡하고, 소비자는 비교 기준이 없다
- 제품 한 줄: 견적 범위 산정 + 공정별 단가 검증 + AI 리스크 탐지

2. Problem
- 왜 지금 필요한가
- 소비자가 겪는 3가지 고통: 불투명한 총액, 묶음 항목, 질문 부족

3. Product Flow
- 4단계 사용자 흐름
- 평수 입력 → 공정/등급 선택 → 단가 검증 → 미팅 질문 생성

4. Demo Highlights
- 15개 공정 데이터
- 3단계 자재 등급
- 공정별 단가 판정 / 총액 판정 / 선택 요약

5. AI Analyzer
- 견적서 텍스트 붙여넣기
- 가격 이상 / 범위 모호 / 누락 항목 / 협상 포인트 추출
- 스트리밍 응답과 로컬 키 저장

6. Why It Works
- 정적 웹앱이라 접근이 쉽다
- 구조화된 가격 데이터와 질문 템플릿
- 실제 협상 행동으로 이어지는 UX

7. Roadmap / Closing
- 견적 비교 엔진: 공정별 차이, 누락 항목, 숨은 추가비, 리스크 점수화
- OCR/PDF 업로드 + 공정 누락 체크리스트 자동 생성
- 협상 시뮬레이터 + 스펙 사전 + 질문 복사 모드
- 내 집 프로필, 결과 저장/공유, 상담 리포트, 수정 견적 이력 관리
- 해커톤 메시지: "견적을 읽는 소비자에게 협상력을 준다"
